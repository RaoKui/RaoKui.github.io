Gradle 学习（二）

发表于 2018-07-23

task

提示有误找不到命令，没有配置环境变量
mac下配置gradle 环境变量：
cd ~
touch .bash_profile
open -e .bash_profile
打开的文件输入下面gradle的变量

最后source .bash_profile

注意
报错：

原因：没有权限。
解决：进入gradle 目录
输入命令：

gradle task指令

gradle assemble:
用于组合项目的所有输出，包含assembleDebug，assembleRelease。执行后打包了release 和debug的包。
单独执行可缩写如：gradle aR(gradle assemble release)

gradle check :
执行检查任务

gralde build ：
类似组合任务，执行了check 和 assemble的所有工作。

gradle clean：
清除中间编译的结果
